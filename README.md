# K-Means Clustering - Customer Segmentation

## Project Overview
This project performs customer segmentation using K-Means clustering on mall customer data to identify distinct customer groups based on annual income and spending patterns.

## Dataset
The dataset contains 200 customers with features including age, gender, annual income, and spending score.

## Key Features
- Data preprocessing and standardization
- Optimal cluster determination using Elbow Method
- K-Means clustering implementation
- Customer segment analysis and profiling
- Business recommendations for each segment

## Results
Identified 5 distinct customer segments ranging from budget-conscious to premium customers, each with specific characteristics and marketing strategies.

## Technologies Used
- Python 3.x
- scikit-learn
- pandas, numpy
- matplotlib, seaborn

## How to Run
1. Clone the repository
2. Install requirements: `pip install -r requirements.txt`
3. Run the Jupyter notebook: `customer_segmentation.ipynb`
